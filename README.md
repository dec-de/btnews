# 睡前消息系视频备份（对外公开）

本仓库用于自动备份部分 B 站视频，以便在视频下架/变更时仍可追溯与检索。
本项目为观众所创建和维护，与相关账号/工作室无任何关系。

## 在 GitHub 下载视频

- 单个视频：进入 `视频汇总/<频道名>/`，打开对应 `.mp4`，点击 “Download raw file / View raw”。
- 批量下载：`git clone` 仓库后在本地查看 `视频汇总/`；如启用了 Git LFS，再执行 `git lfs pull`。
